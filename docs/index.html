VerifyFill

VerifyFill is a Chrome extension that autofills verification codes and email verification links from Gmail.

When you sign in, VerifyFill checks your recent Gmail inbox for messages that look like verification emails. If you are on a website waiting for a code or a confirmation link, it shows a small prompt so you can fill the code or open the link in one click.

Data access

VerifyFill requests read-only access to Gmail (gmail.readonly) and your email address. This is only used to find verification codes and links in recent messages. Email content is processed on your device inside the extension. Nothing is sent to a VerifyFill server. You can sign out or revoke access at any time.

Install

Download the latest release from GitHub, unzip it, then:

  1. Open chrome://extensions
  2. Turn on Developer mode
  3. Click Load unpacked
  4. Select the unzipped verifyfill folder
  5. Click the VerifyFill icon and sign in with Google

Chrome may show a notice that developer extensions are turned on. That is normal for extensions installed outside the Chrome Web Store.
