VerifyFill
Privacy

Open Gmail, copy the code, paste it back. Or don't.

VerifyFill is a Chrome extension that checks your Gmail for verification codes and confirmation links, then offers to drop them into whatever page you're signing into. Read-only. Local. No server.

Get it for Chrome Free · unpackaged install from GitHub releases

How it goes

  1. 1

    Install the extension and sign in with the Google account that gets your codes.

  2. 2

    Go to a site that wants a verification code or tells you to check your email.

  3. 3

    When a match turns up, a small prompt appears on the page. Click it. Done.

What it actually does

  • Codes — numeric and alphanumeric OTPs from recent mail
  • Links — "confirm your email" URLs, if you'd rather click than copy
  • Multi-box fields — the kind with six separate inputs
  • Auto-watch — kicks in when you focus a code field or land on a verify page
  • Recent mail only — inbox, last 15 minutes, up to 20 messages
  • Won't nag twice — remembers codes and links you've already used

Privacy, briefly

VerifyFill needs read-only Gmail access to find codes. It reads your email address to show which account is connected. That's it — no VerifyFill backend, no ads, no selling data. Mail never leaves your machine except through Google's own API.
