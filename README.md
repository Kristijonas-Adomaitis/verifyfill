# VerifyFill

VerifyFill is a Chrome extension that autofills verification codes and email verification links from Gmail.

Sign in with your Google account, then go to a site that asks for a code or tells you to check your email. When a matching message arrives in Gmail, VerifyFill shows a small prompt near the input. Click **Fill** to enter the code, or **Open** for a verification link.

Everything runs in your browser. VerifyFill only reads Gmail — it cannot send or change your email.
