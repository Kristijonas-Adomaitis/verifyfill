VerifyFill

Purpose

The purpose of VerifyFill is to help people sign in to websites faster when a site sends a verification code or email confirmation link to Gmail. Many online services require a one-time code or a link clicked from email before you can log in or create an account. VerifyFill reads those messages from Gmail and offers to enter the code or open the link for you, so you do not have to switch between Gmail and the website manually.

What VerifyFill does

VerifyFill is a Chrome browser extension. After you install it and sign in with Google:

  1. You visit a website that asks for a verification code or tells you to check your email.
  2. VerifyFill looks at recent Gmail messages for codes or confirmation links related to sign-in.
  3. When it finds a match, it shows a small prompt on the page you are using.
  4. You click to autofill the code into the form, or open the verification link.

VerifyFill does not send email, change your inbox, or access Gmail for any purpose other than finding verification codes and links to help you complete sign-in flows.

Why VerifyFill requests Gmail access

VerifyFill needs read-only access to Gmail because verification codes and confirmation links are delivered by email. The extension uses the Gmail API scope https://www.googleapis.com/auth/gmail.readonly and your Google account email address only to:

  • Read recent inbox messages that may contain a verification code or link
  • Show which Google account is connected in the extension

Email content is processed on your device inside the extension. VerifyFill does not operate servers that receive your email. You can sign out in the extension or revoke access in your Google Account at any time.

Install

Download the latest release from GitHub, unzip it, then:

  1. Open chrome://extensions
  2. Turn on Developer mode
  3. Click Load unpacked
  4. Select the unzipped verifyfill folder
  5. Click the VerifyFill icon and sign in with Google

Chrome may show a notice that developer extensions are turned on. That is normal for extensions installed outside the Chrome Web Store.
