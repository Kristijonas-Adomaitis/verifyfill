VerifyFill

Privacy Policy

VerifyFill is a Chrome extension that helps you complete website sign-in when a verification code or email confirmation link is sent to Gmail. The application connects to your Google account with your permission, reads recent Gmail messages to find verification codes or links, and offers to autofill them on the website you are using.

Application purpose

The purpose of this application is to reduce manual copying between Gmail and websites during login, account creation, and two-step verification. When a website asks for a one-time code or tells you to click a link in your email, VerifyFill finds that message in Gmail and helps you apply it on the current page.

Application functionality

VerifyFill provides the following features to users:

  • Sign in with Google to connect a Gmail account
  • Scan recent inbox messages for verification codes (for example, 6-digit OTP codes)
  • Scan recent inbox messages for email verification and sign-in links
  • Display a prompt on the active browser tab to fill a code or open a link
  • Let the user autofill a code into a form or open a verification link in one click

How this app uses Google user data

VerifyFill uses Google Sign-In and the Gmail API only for the features listed above. This application requests the following Google OAuth scopes:

  • https://www.googleapis.com/auth/gmail.readonly — to read recent Gmail messages and detect verification codes or confirmation links
  • https://www.googleapis.com/auth/userinfo.email — to show which Google account is connected

Google user data is used only to provide verification autofill. Email content is processed locally in the user’s browser inside the Chrome extension. VerifyFill does not sell Google user data, does not use it for advertising, and does not upload inbox content to a VerifyFill server. Users can sign out in the extension or revoke access in their Google Account permissions at any time.

Install

Download the latest release, unzip it, open chrome://extensions, enable Developer mode, click Load unpacked, and select the extension folder.
