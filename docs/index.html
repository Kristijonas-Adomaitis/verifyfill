VerifyFill

VerifyFill is a Chrome extension that autofills verification codes and email verification links from Gmail.

Sign in with your Google account, open a site that asks for a code or tells you to check your email, and VerifyFill will offer to fill the code or open the link when the message arrives.

Everything runs in your browser. VerifyFill only reads Gmail — it cannot send or change your email.

Install

Download the latest release from GitHub, unzip it, then:

  1. Open chrome://extensions
  2. Turn on Developer mode
  3. Click Load unpacked
  4. Select the unzipped verifyfill folder
  5. Click the VerifyFill icon and sign in with Google

Chrome may show a notice that developer extensions are turned on. That is normal for extensions installed outside the Chrome Web Store.
